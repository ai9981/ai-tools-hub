CodeRabbit

PR レベルで自動コードレビューを行うプラットフォーム。問題点の指摘と修正案を提案します。

導入と設定

  1. CodeRabbit のアカウントを作成し、GitHub/GitLab 連携を行います。
  2. リポジトリ単位でルールとスコーピングを設定します（除外パス、CI 連携など）。

主な機能とワークフロー

  • PR が作成されると自動で静的解析 + AI レビューを実行。
  • 修正提案をコメントとして残し、必要に応じて修正 PR を自動生成できます。

運用上の注意

  • 誤検出（false positive）対策として学習フィードバックやホワイトリストを設定する。
  • 自動修正を有効化する場合は段階的ロールアウト（小規模リポジトリで確認）を推奨。
