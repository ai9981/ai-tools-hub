Qodo (formerly Codium)

テストコード自動生成とロジック検証に特化したツール。品質保証プロセスの自動化を助けます。

主な機能

  • ユニットテスト生成、ファジング テスト、ロジック検証。
  • CI パイプラインへの自動統合（PR 時にテストを生成・検証）。

使い方の例

# CLI（疑似）
qodo generate-tests --path src/utils --framework jest

# CI で実行する例
# on: pull_request
# run: qodo ci --repo $GITHUB_WORKSPACE

Settings & Integration

# Qodo API key
export QODO_API_KEY="your_api_key"

# curl 例
curl -H "Authorization: Bearer $QODO_API_KEY" https://api.qodo.example/v1/generate -d '{"path":"src"}'
