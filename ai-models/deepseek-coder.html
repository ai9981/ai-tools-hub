DeepSeek-Coder-V2

コスト効率に優れた軽量モデル。多くの開発ツールでバックエンドとして採用されます。

導入

プロバイダが提供する SDK や REST API に従って接続します。低レイテンシと低コストが特徴。

ユースケース

  • 補完や簡易なリファクタリング。高頻度なAPI呼び出しが必要な場面に向く。
  • CI/CD の軽量な自動化タスク（テスト生成、Lint 修正提案等）。

トラブルシューティング

  • 性能が不足している場合は、より大きなモデルにフェイルオーバーする設計を考える。
  • 大量リクエスト時のレート制限やスロットリングを確認する。

Sample Prompts（実例）

  1. テストコード生成 (日本語): 
    Jest を使って src/utils/parseCsv.js のユニットテストを生成してください。悪意のある入力や欠損データのケースも含めてください。
    Unit test generation (English): 
    Generate unit tests for src/utils/parseCsv.js using Jest. Cover edge cases for malformed input.
  2. 簡易リファクタリング (日本語): 
    このヘルパー関数のより単純な実装を提案し、更新版を示してください。
    Simple refactoring (English): 
    Suggest a simpler implementation for this helper function and provide an updated version.
  3. CI 自動化タスク (日本語): 
    PR 時にテストと Lint を実行し、結果を Slack に通知する GitHub Actions ワークフローを作成してください。
    CI automation task (English): 
    Write a GitHub Actions workflow that runs tests and lints on PRs, and posts status to Slack.

Cost & Performance（目安）

項目目安備考
コスト低低コストで高いスループットが得られる
レイテンシ低即時応答が必要な場面に適する
得意分野補完・自動化タスク高頻度呼び出しに向く

Settings & SDK（よくある設定）

例: 環境変数と簡易的なエンドポイント利用

# 環境変数
export DEEPSEEK_API_KEY="your_api_key_here"
export DEEPSEEK_ENDPOINT="https://api.deepseek.example/v1"

# curl 例
curl -X POST "$DEEPSEEK_ENDPOINT/generate" -H "Authorization: Bearer $DEEPSEEK_API_KEY" -d '{"input":"Generate unit tests for parseCsv"}'
