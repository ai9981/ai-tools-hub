Cline

VS Code 上で自律プレイブックを実行するエージェント拡張。ターミナル実行やブラウザ確認などを自動化します。

インストール

  1. VS Code の拡張マーケットで "Cline" を検索してインストールします。
  2. 必要に応じて API キーやクラウド連携情報を設定します。

基本的な使い方

  1. エージェントに実行したいプレイブック（手順）を与える。
  2. ターミナルコマンド、ブラウザ確認、スクリーンキャプチャなどを自動化できます。

コマンド例

// 例: プレイブックの例
// 1. Run tests
// 2. If tests pass, build and deploy to staging

トラブルシューティング

  • エージェントがシステムコマンドを実行できない: 権限設定を見直す。
  • 期待通りに動かない: プレイブックを小さく分けて再試行する。
