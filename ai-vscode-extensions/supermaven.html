Supermaven

超大コンテキスト（1Mトークン）を扱う補完特化の拡張。高速な補完が特徴です。

インストール

  1. Visual Studio Code を開く → 拡張パネルで "Supermaven" を検索してインストール。
  2. 拡張のインストール後、必要に応じて設定でコンテキストサイズやモデル接続を調整します。

アカウントと接続

クラウドサービスの API キーが必要な場合は、拡張の設定または環境変数でキーを指定します。

使い方（よく使う操作）

  1. 大きなリポジトリを開いた状態で補完を試すと、プロジェクト全体を考慮した提案が得られます。
  2. 設定でコンテキストウィンドウの範囲や優先モデルをカスタマイズしてください。

トラブルシューティング

  • 応答が遅い場合はモデル接続先やネットワーク状況を確認する。
  • 大きなコンテキストでメモリ制限に達することがあるため、必要に応じて設定で制限をかける。
