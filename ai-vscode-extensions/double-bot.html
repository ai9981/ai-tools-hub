Double.bot

シンプルかつ高速なコーディング体験を追求する新進の拡張機能。

インストール

  1. VS Code の拡張マーケットで "Double.bot" を検索してインストールします。
  2. 拡張の設定で応答速度や表示スタイルを調整します。

使い方

  1. 補完やスニペットを素早く呼び出して軽快にコーディングします。
  2. 設定で好みのショートカットを割り当てると更に生産性が上がります。

コマンド例

// 補完の呼び出しやスニペットの使用方法は拡張のドキュメントを参照してください

トラブルシューティング

  • 応答が期待より遅い場合は、拡張の設定でオフラインモードやモデルのプロバイダ設定を見直す。
  • スニペットが予期せぬ場所で挿入される場合はキーマッピングを確認。
