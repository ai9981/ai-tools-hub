Amazon Q

AWS 向けの開発・診断に特化した拡張。レガシー近代化や脆弱性検査の自動化に強みがあります。

インストール

  1. VS Code の拡張マーケットで "Amazon Q" を検索してインストールします。
  2. AWS 認証情報（環境変数やプロファイル）を設定して拡張に接続します。

使い方（よく使う操作）

  1. 既存の AWS プロジェクトを開き、拡張のスキャン機能で脆弱性や最適化案を確認します。
  2. レガシー Java プロジェクトのアップグレード提案（例: Java 8 -> 17）を自動提案することがあります。

コマンド例

// 例: スキャンの実行（CLIが提供されている場合）
// aws-q scan --project .

トラブルシューティング

  • AWS 認証エラー: プロファイル/環境変数を確認し、必要な IAM 権限があるか確認する。
  • スキャンのエラー: 大規模プロジェクトでは時間がかかるため、範囲を絞って試す。
