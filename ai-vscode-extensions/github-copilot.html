GitHub Copilot

GitHub公式の補完エンジン。Copilot Workspace等の連携でPR作成支援も可能。

インストール

  1. VS Code の拡張マーケットで "GitHub Copilot" を検索してインストールします。
  2. インストール後、GitHub アカウントでサインインし、必要な権限を許可します。

設定とサインイン

拡張は GitHub アカウントと連携します。組織アカウントを使う場合は管理者の許可が必要になる場合があります。

使い方（よくある操作）

  1. エディタに関数の一部を書くと自動補完が提案される。
  2. 複数の候補がある場合は Tab で選択、提案を受け入れることができます。

コマンド例

// 補完の提案を受けたい場合は
// Type a comment describing the function and start typing the function signature.

トラブルシューティング

  • サインインできない: ブラウザで GitHub にログインしているか確認し、拡張を再ログイン。
  • 候補が出ない: Copilot の有効化設定を確認、言語とファイル種別が対応しているか確認する。
