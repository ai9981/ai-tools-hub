Cursor

AIネイティブなコードエディタ。ここではインストール、アカウント、基本的な使い方を解説します。

スクリーンショット

このリポジトリでは実際のスクリーンショット画像を含めていません。

インストール

  1. 公式サイトにアクセス: https://www.cursor.dev
  2. 提供形式を確認: デスクトップアプリ / Web / VS Code 互換拡張 のいずれか。
  3. デスクトップならインストーラをダウンロードして実行、拡張なら VS Code の拡張としてインストールします。
  4. インストール後、アプリを起動して次の手順へ進みます。

詳細インストール手順（Windows / VS Code 拡張例）

  1. VS Code を開く → 左ペインの拡張機能アイコンをクリック。
  2. 検索ボックスに "Cursor" と入力し、公式拡張を選んで "Install" をクリック。
  3. 拡張がインストールされたら VS Code を再起動します。
  4. 拡張パネルの設定で API キーやアカウント連携を行います（必要に応じて GitHub OAuth を使います）。

コマンドラインでの補助（例: プロジェクトルートで操作する場合）:

cd path\to\your\project
# プロジェクト依存のインストール（例）
npm install
# エディタでプロジェクトを開く（Windows の場合）
code .

アカウント作成・ログイン

  1. メールアドレスでサインアップ、または OAuth（GitHub/Google）を利用。
  2. 確認メールが届いたら認証を完了。
  3. アプリにログインし、必要ならプロジェクトを作成します。

基本的な使い方

  1. 新規プロジェクトを作成するか、既存のリポジトリを開く。
  2. サイドバーの "AI" パネルを開き、自然言語で指示を入力（例: "React の新しいページを作って"）。
  3. 生成されたコードはプレビューしてから適用。必要に応じて編集しコミットします。
  4. リファクタリングやテスト生成も同様に指示で実行可能。

実例コマンド

// 例: ファイル生成の指示
Create a new Next.js page at /pages/about.js with a title and two paragraphs.

実践ガイド — よく使う操作とコマンド例

1) 新しいコンポーネントを自然言語で作る

  1. AI パネルに次のように入力します:
    Create a new React component named ProfileCard in src/components with props: name, avatarUrl, bio.
  2. 生成内容を確認し、保存（Ctrl+S）してコンポーネントを利用します。

2) リファクタリングを依頼する

  1. 編集したいファイルを開き、AI パネルに「Refactor this file to use hooks instead of classes」と入力。
  2. 提案をプレビューして問題なければ Apply。

3) テストの自動生成

Generate unit tests for src/utils/formatter.js using Jest.

生成されたテストを実行:

npm test -- src/utils/formatter.test.js

4) Git と連携して変更をコミットする

  1. 生成内容に満足したらターミナルで次を実行します:
    2. git add .
git commit -m "chore: add ProfileCard component (generated by Cursor)"
git push origin main

トラブルシューティング

  • API キーの設定が反映されない: 拡張の設定を開いてキーを再入力、VS Code を再起動。
  • 生成が期待と異なる: 指示をより具体的に（ファイルパス、スタイル、テスト要件などを明示）。
  • プライバシー懸念: ローカルモデル接続やオフラインモードがあればそちらを利用。

参考: 公式サイトを常に確認してください — 機能やUIは頻繁に更新されます。
