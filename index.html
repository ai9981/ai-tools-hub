1. AI Native Editors & Environments

Cursor

STANDARD

VS Codeを完全に置き換える勢いのAIエディタ。自然言語でのファイル生成やバグ修正の精度が極めて高い。

#Composer #CodebaseIndex

Windsurf

NEW GEN

Codeium提供。エージェントが人間の意図を「Flow」として解釈し、自律的にコーディングを進める。

#Flow #AgenticIDE

Void

OSS

Cursorのオープンソース代替を狙うプロジェクト。完全にコントロール可能なAIエディタ環境を提供。

#OpenSource #CursorAlt

Replit Agent

CLOUD IDE

環境構築不要。スマホやブラウザから「Webアプリを作って」と命じるだけで完成・公開まで行う。

#Deployment #ZeroConfig

2. VS Code Extensions & Plugins

Supermaven

ULTRA FAST

100万トークンのコンテキスト窓と、他を圧倒する「補完スピード」が特徴。開発の思考を妨げない。

#1M_Context #SpeedKing

GitHub Copilot

STANDARD

GitHub公式。リポジトリ管理と密結合した「Copilot Workspace」によりPR作成の自動化が加速。

#Workspace #Enterprise

Cline

AGENTIC

VS Code上で「自律エンジニア」として振る舞う。ターミナル実行、ブラウザ確認などをAIが行う。

#Autonomous #ClaudeDev

Amazon Q

AWS OPTIMIZATION

AWS上のアプリ開発、脆弱性診断、レガシーシステムの近代化（Java 8→17等）に特化した強み。

#AWS #CodeScan

Double.bot

FAST GROWTH

VS Codeでのコーディング体験を極限までシンプルかつ高速にすることに特化した新興拡張機能。

#UserExperience #FastAI

Continue.dev

OPEN SOURCE

任意のLLMと接続可能なオープンな土台。ローカルLLMを用いた完全オフライン開発も可能。

#CustomLLM #OfflineReady

3. Leading Models (The Engines)

Claude 3.5 Sonnet / 4

LOGIC KING

#Reasoning #Anthropic

Kimi (Moonshot AI)

#LongContext #LargeScale

DeepSeek-Coder-V2

HIGH EFFICIENCY

非常に安価かつ軽量でありながら、性能はClaudeやGPTに肉薄。多くのツールの裏側で採用。

#MoE #CostPerformance

Qwen2.5-Coder / 3

TOP OSS

#OpenSource #StateOfTheArt
