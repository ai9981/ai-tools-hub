Aider / Plandex

ターミナル中心の対話型エージェント。ファイル編集、コミット、PR 作成までのループが高速です。

インストール

# Homebrew (macOS)
brew install aider

# npm (Cross-platform)
npm i -g aider-cli

使い方（CLI 例）

# レポジトリでタスクを実行
aider run --task "Fix failing tests" --auto-commit

# 対話式で指示を出す
aider interactive

注意点

  • 自動コミットや自動マージの設定は慎重に。まずは --dry-run やテストブランチで試す。
  • CI 認証・トークンの権限を適切に設定する（最小権限の原則）。
